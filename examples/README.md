# Badge Examples

This directory contains example badges generated with the Vibe Coding Badge Generator.

## Original Template

![Original Template](original-template.png)

The original template that inspired this project - a simple blue badge showing "Vibe Coded! Opus 4.5".

## How to Generate Your Own

1. Visit the [Badge Generator](https://danielrosehill.github.io/Vibe-Coded-Icons/)
2. Customize your badge with:
   - Model company and name
   - Your preferred color gradient
   - Badge style (Display, Banner, Square, or QR Code)
   - Optional license and human name
3. Download as PNG or SVG

## Usage

After generating your badge, add it to your README:

```markdown
![Vibe Coded](./path/to/your-badge.png)
```

Or use it inline:

```markdown
This project was built with AI assistance ![Vibe Coded](./badge.png)
```
